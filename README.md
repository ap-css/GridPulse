# GridPulse ⚡
> Feel the pulse of your power grid.

**GridPulse** is a premium, real-time power outage tracker for Karnataka (BESCOM focus). Built with a focus on UI/UX, it features a stunning neon glassmorphism design, interactive cursor effects, and weather-based risk assessment.


## 🚀 Features
- **Neon Glassmorphism UI**: Frosted glass cards with vibrant neon accents.
- **Live Outage Feed**: Searchable and filterable data for scheduled BESCOM outages.
- **Weather-Based Risk**: Integration with Open-Meteo API to calculate outage probability.
- **Dynamic Theme**: Sleek Dark and Light modes with persistent state.
- **Telegram Alerts**: Frontend subscription logic for proactive notifications.
- **Interactive Stats**: Clickable stat cards to quickly filter the feed.
- **Clean Cursor Experience**: Smoothed neon-dot cursor trailing.

## 🛠️ Built With
- **HTML5**: Semantic structure.
- **CSS3**: Custom properties, Glassmorphism, Neon effects, Keyframe animations.
- **JavaScript (Vanilla)**: Data filtering, Theme engine, Weather API integration.
- **Open-Meteo API**: Live weather data sourcing.

## 📦 How to Run Locally
1. Clone the repository.
2. Open `index.html` in your browser (use a live server extension if possible for icons/SVG rendering).
3. Or use `npx serve`:
   ```bash
   npx serve
   ```

## 🌏 Deployment
The app is designed to be hosted on **Vercel** or **GitHub Pages**. For Vercel, simply connect your GitHub repo and hit deploy.
